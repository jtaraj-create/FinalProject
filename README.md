\# Library Book Management System



\## Overview



This project is a Library Book Management System developed using HTML, CSS, JavaScript, PHP, and MySQL. It allows users to manage a collection of books through a simple web interface connected to a database.



\## Features



\* View all books

\* Add new books

\* Edit existing books

\* Delete books

\* Search books by title, author, or ID

\* Filter books by category

\* Store all data in a MySQL database



\## Technologies Used



\* HTML5

\* CSS3

\* JavaScript

\* PHP

\* MySQL

\* XAMPP



\## Installation



1\. Download or clone the project.



2\. Copy the project folder into:



&#x20;  C:\\xampp\\htdocs\\



3\. Start Apache and MySQL from XAMPP.



4\. Open phpMyAdmin and create a database named:



&#x20;  library\_db



5\. Import the SQL file located in the `database` folder.



6\. Open the project in your browser:



&#x20;  http://localhost/FinalProject/



\## Project Structure



\* `api/` – API endpoints for database operations

\* `assets/` – CSS, JavaScript, and other resources

\* `config/` – Database configuration files

\* `database/` – SQL database file

\* `index.html` – Main page

\* `add-book.html` – Add a new book

\* `edit-book.html` – Edit a book

\* `delete-book.html` – Delete a book



\## Functionality



The application implements full CRUD operations:



\* Create – Add a new book

\* Read – Display books from the database

\* Update – Modify existing book information

\* Delete – Remove books from the database



All changes are stored permanently in the MySQL database and remain available after refreshing or reopening the application.
